Help for Upcoming view

In this view you will see a list with upcoming events. At the top of the list the time period will be shown. The start date will always be the date of today and the end date depends on the time span specified by the administrator.

On this page you will find the following hyperlinks:
  • Event title will open the Event Report with all event details, or, if you have edit rights, it will open the Event Edit window.
  • Download button will download a text file with all upcoming events shown on this page.

For an explanation of the Navigation bar buttons, select the General Help button in the right upper corner.
